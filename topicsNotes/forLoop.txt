for loop can be used in a traditional way but also it can be used in new syntax
for .. in loop

for item in items
do
	do something
done

you can also loop for output of some command in the same way just place
$(command) in place of items

for looping in array traditional way can be used

and for in loop can be used like

for item in ${arrname[*]}
do
	do something
done
