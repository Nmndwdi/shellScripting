this project can be executed by sudo or root users only
this project is for creating new users with initial password and forcing them to change password on their first login , and adding the newly created user to a specific group , userName and groupName will be provided as arguments for the script file , this task can be performed by executing the createUser file

in this project we will also be able to edit the group of any user and delete any user by executing editUser file and deleteUser file

createUser : you can create multiple users by giving userName and groupName one after another if you does not want user to add in a group give 'null' in place of group name

e.g. : ./createUser.sh userName1 groupName1 userName2 groupName2 userName3 null
userName4 groupName4 userName5 null
